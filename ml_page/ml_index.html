K-Nearest Neighbors

KNN works by finding the distance between a query and all the examples in the data, selecting the specified number examples (K) closest to the query, then votes for the most frequent label (in the case of classification) or averages the labels (in the case of regression).

Logistic Regression

Logistic regression is a type of linear regression that predicts the probability of a binary event through a logit function (Maximum Likelihood Estimation/MLE). MLE is a likelihood maximization method, which sets the mean and variance as parameters most likely to produce the observed data.
